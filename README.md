# Power BI Financial Dashboard

This project is a simple and beginner-friendly Power BI dashboard built using the **Microsoft Financial Sample** dataset.

## 📊 Dashboard Features
- Total Sales (Card)
- Sales by Country (Bar Chart)
- Sales Trend Over Time (Line Chart)
- Sales by Segment (Donut Chart)
- Country Slicer for filtering

## 🗂 Files Included
- **Financial Dashboard.pbix** – Power BI dashboard file
- **Financial Sample.xlsx** – Dataset used for the dashboard
- **README.md** – Project documentation

##  How to Use
1. Download the `.pbix` file
2. Open it in **Power BI Desktop**
3. Explore the visuals and interact with the slicers

## 📁 Dataset Source
Microsoft Financial Sample Dataset  
(Official Microsoft sample used for learning and practice)
